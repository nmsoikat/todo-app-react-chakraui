### Todo Application

```
npm install
npm start
```

### Tools:

- [React](https://reactjs.org/)
- [ChakraUI](https://chakra-ui.com/)

### Features:

- Add task.
- Add with priority.
- Sort with priority.
- Remove task.
- Edit and update.
- Show toast message if user try to add empty title.
- Show message if there have not any task to do.
- Local Storage.
- Dark Mode.
- Responsive.
