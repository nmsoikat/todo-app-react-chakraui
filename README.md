### Todo Application

```
npm install
npm start
```

### Tools:

- [React](https://reactjs.org/)
- [ChakraUI](https://chakra-ui.com/)

### Features:

- Add task.
- Remove task.
- Show toast message if user try to add empty value.
- Show message if there have not any task to do.
- Local Storage.
- Dark Mode.
- Responsive
